# How to run

1. Download [FreeImage](http://downloads.sourceforge.net/freeimage/FreeImage3180.zip)

2. Place the zip file **next to** the project directory. E.g.
  ```
  dir/
    15618-project
    FreeImage3180.zip
  ```

3. Build FreeImage
  ```sh
  unzip FreeImage3180.zip
  cd FreeImage
  make
  ```
